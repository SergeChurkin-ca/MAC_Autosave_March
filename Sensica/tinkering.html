Sensica™ Urine Output System

The Sensica™ Urine Output System is an automated urine output monitoring system for continuous monitoring of urine output and core bladder temperature when connected to a temperature-sensing Foley catheter.



Real-time urine output data at your fingertips.


* This capability is subject to the Sensica Urine Output System being connected to the EMR via BD’s EMR Interoperability solution which is available for an additional fee.

Sensica™ Urine Output System:

  • Continuously monitors urine output and core bladder temperature when connected to a temperature sensing Foley catheter.
  • Automatically transmits data into the patient's electronic medical record (EMR).*
  • Features a touch screen display with trended data graphs, customizable patient settings, Foley duration tracking, and default weight normalized urine output alarms that align with internationally recognized Acute Kidney Injury identification guidelines.
